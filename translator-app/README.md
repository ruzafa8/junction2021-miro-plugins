# Miro translator app

## Requirements

You need an authentication key for [DeepL's API](https://www.deepl.com/pro-api?cta=header-pro-api).

Create a file named `src/insecure_env.js` containing the key:

```js
const env = {
    DEEPL_API_KEY: "your-authentication-key"
}
```
