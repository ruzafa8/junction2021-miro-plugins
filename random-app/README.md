# Miro random app

## Requirements

You need an API key for [the RANDOM.ORG API](https://api.random.org/dashboard).

Create a file named `src/insecure_env.js` containing the key:

```js
const env = {
    RANDOM_ORG_API_KEY: "your-api-key"
}
```
